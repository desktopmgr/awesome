### Awesome Key Bindings

|                  |                 |
| :--------------: | :-------------: |
|    Super + s     |    Show Help    |
|   Super + 1-9    | Change desktops |
|   Super + Left   |  View Previous  |
|  Super + Right   |    View Next    |
|   Alt + Escape   |     Go Back     |
|    Alt + Tab     |    View Next    |
| Alt + Shift +tab |  View Previous  |
|     Alt + j      |   Focus Next    |
|     Alt + k      | Focus Previous  |
|    Super + j     |   Focus Down    |
|     Super +k     |    Focus Up     |
|    Super + h     |   Focus Left    |
|     Super +l     |   Focus Right   |
|   Super + Down   |   Focus Down    |
